# Task Management System

## Overview

A simple full-stack Task Management System built using React, Node.js, Express.js and MySQL.

## Features

- User Registration
- User Login
- JWT Authentication
- Create Task
- View Tasks
- Update Task
- Delete Task

## Tech Stack

### Frontend

- React
- Axios
- React Router DOM

### Backend

- Node.js
- Express.js
- MySQL
- JWT
- bcrypt

## Installation

### Backend

```bash
npm install
npm start
```

### Frontend

```bash
npm install
npm run dev
```
